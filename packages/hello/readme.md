# NODE typescript template

## Description

Template nodejs typescript on workspace

## Contributing

1. npm run start -  Start project
2. npm run dev - Start project with ts-node (dev only)
3. npm run compile - Build ts

## Production

```bash
npm i --production
npm start
```
